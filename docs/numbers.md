# Type numbers 

Hold left hand on peace gesture,and
use the right hand to type the number

`peace + palm` = `1`
`peace + fist` = `2`
`peace + ok` = `3`
`peace + one` = `4`
`peace + two_up` = `5`
`peace + peace` = `6`
`peace + like` = `7`
`peace + four` = `8`
`peace + three` = `9`
`peace + three2` = `0`
