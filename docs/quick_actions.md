# Quick Actions that you can perform using only Right hand


## Pause/activate app
`dislike + dislike`
Note: even when paused mouse sill held by the app

## press escape button
`fist` = `esc`

## move window focus (right/left)
`three2` = `move focus right`

## resize window
`stop_inverted` = `resize window left`
`two_up_inverted` = `resize window right`

## Close the current active window
`dislike` = `SUPER+q`

## toggle fullscreen
`palm` = `toggle fullscreen`

## App menu (rofi)
`call` = `app launcher (wofi)`

---
>>>>>>>> { media control } <<<<<<<<<<<<

## play / pause
`stop` = `play / pause`

## Volume control
`one` = `volume up`
`two_up` = `volume down`

## Mute
`mute` = `mute / unmute`

## Track control
`peace` = `next track`
`inverted` = `previous track`

## Seek
`three` = `skip forward`
`four` = `skip backward`
