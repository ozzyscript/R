# Simulate Mouse functionality 

### Note: see `g_consts.py` file to adjust

## Clicks
`fist + ok` = `left click`
`fist + three2` = `right click`
`fist + ok` = `middle click`  
`fist + fist` = `double left click`

## Cursor
Note: cursor move gestures are in mouse.py
below is their combo gestures 

### Main 
`fist + one` => up
`fist + two_up` => down
`fist + three` => left
`fist + four` => right

### Diagonals
`fist + stop` => right up
`stop_inverted` => left up
`fist + dislike` => right down
`fist + like` => left down

### Scroll
`fist + peace` = `scroll up`   # scroll up
`fist + inverted` = `scroll down`  # scroll down
