# Type letters

Hold left hand on stop gesture, and
use the right hand to type the char/symbol

`stop + palm` = `a`
`stop + fist` = `b`
`stop + ok` = `c`
`stop + one` = `d`
`stop + two_up` = `e`
`stop + peace` = `f`
`stop + like` = `g`
`stop + four` = `h`
`stop + three` = `i`
`stop + three2` = `j`
`stop + rock` = `k`
`stop + call` = `l`
`stop + mute` = `m`
`stop + dislike` = `n`
`stop + two_up_inverted` = `o`

---

`stop_inverted + palm` = `p`
`stop_inverted + fist` = `q`
`stop_inverted + ok` = `r`
`stop_inverted + one` = `s`
`stop_inverted + two_up` = `t`
`stop_inverted + peace` = `u`
`stop_inverted + like` = `v`
`stop_inverted + four` = `w`
`stop_inverted + three` = `x`
`stop_inverted + three2` = `y`
`stop_inverted + rock` = `z`
