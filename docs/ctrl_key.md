# Simulate CTRL Key

Hold left hand on `like` gesture, and
use the right hand to type the char/symbol

`like + palm` = `CTRL+a`  # select all
`like + fist` = `CTRL+c`  # copy
`like + ok` = `CTRL+v`    # paste
`like + one` = `CTRL+SHIFT+c`  # copy terminal Linux
`like + two_up` = `CTRL+SHIFT+v`  # paste terminal Linux
`like + stop` = `CTRL+z`  # Undo
`like + stop_inverted` = `CTRL+SHIFT+z`  # Redo
