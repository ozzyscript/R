# Type Chars/Symbols

Hold left hand on peace gesture,and
use the right hand to type the char/symbol

`peace + rock` = `@`
`peace + call` = `#`
`peace + mute` = `%`
`peace + dislike` = `&`
`peace + stop` = `"`
`peace + stop_inverted` = `'`
